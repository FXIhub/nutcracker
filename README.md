# Nutcracker
A collection of software tools for validation of flash xray imaging reconstructions packaged as a Python library.

## Authors:
Benedikt Daurer     
Louis Doctor     
Tomas Ekeberg     

## Dependencies:
Following packages are required to run Nutcracker:
* numpy    
* scipy    
* spimage    
* condor    
* h5py
